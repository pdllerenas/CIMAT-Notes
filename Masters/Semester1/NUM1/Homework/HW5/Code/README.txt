# Homework 5

## How to execute (assuming gcc + Linux kernel 6.14.0-28-generic):

### Problem 1:
To compile and run the first problem, use `make run-p1 ARGS="t5a/<matrix-filename>.txt t5a/<vector-filename>.txt <m-eigenvalues> <TOLERANCE> <MAX_ITERATION>"`.

### Problem 2:
To compile and run the second problem, use `make run-p2 ARGS="t5a/<matrix-filename>.txt t5a/<vector-filename>.txt <m-eigenvalues> <TOLERANCE> <MAX_ITERATION>"`.
