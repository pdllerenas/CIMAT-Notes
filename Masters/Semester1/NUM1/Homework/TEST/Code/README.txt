# Examen parcial 1

## How to execute (assuming gcc + Linux kernel 6.14.0-28-generic):

### Problem 1:
To compile and run the first problem, use `make run-p1 ARGS="<tolerance> <max-iter>"`.

### Problem 2:
To compile and run the second problem, use `make run-p2 ARGS="<file1-out> <file2-out>"`.

### Problem 3:
To compile and run the third problem, use `make run-p3 `.

### Problem 4:
To compile and run the fourth problem, use `make run-p4"`.
